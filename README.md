# SafeHome360

A smart home IoT security dashboard built with React. Monitors and visualizes security data from IoT devices in real time.

## Project Structure

```
safehome360/
├── iot-security-dashboard/   # React frontend dashboard
│   ├── src/
│   │   ├── App.js
│   │   ├── Codes.js           # Main IoT Security Dashboard component
│   │   └── index.js
│   └── public/
├── package.json
└── test.py
```

## Getting Started

### Install dependencies

```bash
cd iot-security-dashboard
npm install
```

### Run the app

```bash
npm start
```

Opens at [http://localhost:3000](http://localhost:3000)

### Build for production

```bash
npm run build
```

## Tech Stack

- React 19
- Lucide React (icons)
- CSS3
